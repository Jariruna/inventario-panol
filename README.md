# Sistema de Gestión de Pañol (PañolCentral)

Este es un sistema web ligero para el control de inventarios, movimientos de herramientas y reportes operativos, diseñado para mejorar la eficiencia en la gestión de insumos en el pañol.

## 🚀 Características principales
- **Gestión de Movimientos:** Registro de entradas y préstamos de herramientas.
- **Persistencia de Datos:** Uso de `localStorage` para mantener el inventario disponible sin necesidad de servidor.
- **Filtros Dinámicos:** Búsqueda y filtrado por estado y tipo de movimiento.
- **Exportación:** Generación de reportes en formato CSV compatible con Excel.
- **Notificaciones:** Sistema de alertas flotantes para una mejor experiencia de usuario.

## 🛠️ Tecnologías utilizadas
- HTML5 / CSS3 (Diseño responsivo).
- JavaScript (Lógica de negocio y manipulación del DOM).
- Git (Control de versiones).

## 📋 Cómo utilizar
1. Clona este repositorio en tu equipo.
2. Abre el archivo `inventario.html` en tu navegador favorito.
3. ¡Empieza a registrar tus movimientos de pañol!
